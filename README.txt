battery-charge-limits is a laptop-mode-tools module for contol values
`charge_control_start_threshold` and `charge_control_end_threshold` at
`/sys/class/power_supply/BAT[0-9]*` .

`battery-charge-limits` -> `/etc/laptop-mode/modules/battery-charge-limits`
`battery-charge-limits.conf` -> `/etc/laptop-mode/conf.d/battery-charge-limits.conf`
